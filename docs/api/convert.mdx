## Overview

The `POST /api/convert` endpoint provides synchronous, real-time extraction of web domains into clean, LLM-ready Markdown. The SaaS backend intelligently routes your request through native HTTP parsing, static DOM manipulation, or headless Chrome browser clusters depending on the target's JavaScript reliance.

**Base URL:** `https://api.2md.traylinx.com`

<Note>
**Enterprise Streaming:** This endpoint streams real-time NDJSON progress logs natively over HTTP. If your client terminates the connection early, the cluster immediately kills the headless browser instance to preserve your rate limits.
</Note>

## Authentication

All standard conversions (HTML to Markdown) do not require an active SwitchAI token. Rate limits apply based on origin IP.

## Request

`POST /api/convert`

**Content-Type:** `application/json`

### Body Parameters

<ParamField body="url" type="string" required>
  The target URL to process. The cluster automatically bypasses standard geo-blocks and captchas.
</ParamField>

<ParamField body="method" type="string" default="auto">
  Explicitly force an extraction engine. Overrides the auto-router.
  - `auto` — Intelligently escalates through extraction tiers (default).
  - `native` — High speed fetch for raw MD targets.
  - `static` — Cheerio/Readability pipeline. No JS execution.
  - `browser` — Forces a headless Puppeteer instance for SPAs.
</ParamField>

<ParamField body="format" type="string">
  Response format serialization.
  - `json` — Returns a structured JSON payload with tokens and metadata (default for API).
  - `markdown` — Returns just the raw string block.
  - `stream` — Streams NDJSON logs sequentially before delimiter `__JSON__`.
</ParamField>

<ParamField body="downloadImages" type="boolean" default={true}>
  Proxies embedded images, caching them on the 2md CDN.
</ParamField>

<ParamField body="waitMs" type="number" default={10000}>
  Time to permit React/Vue hydration when `method: browser` is invoked.
</ParamField>

## Response

### 200 — Success Payload (`format=json`)

```json
{
  "success": true,
  "url": "https://example.com/blog",
  "markdown": "# Example Blog\n\nThis content is clean...",
  "cache": "miss",
  "method": "browser",
  "tokens": {
    "html": 45100,
    "md": 1024
  },
  "metadata": {
    "title": "Example Blog",
    "byline": "Author Name"
  }
}
```

<ResponseField name="tokens" type="object">
  Native metric comparing the bloat of the raw DOM (`html`) against the optimized payload (`md`). Use this to log vector DB savings in your pipeline.
</ResponseField>

## Production Examples

<CodeGroup>

```bash cURL
curl -X POST https://api.2md.traylinx.com/api/convert \
  -H "Content-Type: application/json" \
  -d '{"url": "https://stripe.com/docs"}'
```

```python Python (Requests)
import requests

response = requests.post(
    "https://api.2md.traylinx.com/api/convert",
    json={"url": "https://stripe.com/docs", "waitMs": 5000}
)
data = response.json()
print(f"Extraction saved {data['tokens']['html'] - data['tokens']['md']} tokens!")
```

```javascript Node.js (Fetch)
const response = await fetch('https://api.2md.traylinx.com/api/convert', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://stripe.com/docs', format: 'markdown' })
});

const text = await response.text();
console.log(text);
```

</CodeGroup>
