## Overview

The `POST /api/crawl` endpoint is engineered for massive corpus ingestion. It handles topology mapping, link crawling, and asynchronous batch worker assignment so you don't have to manage local connection timeouts.

**Base URL:** `https://api.2md.traylinx.com`

## Request

`POST /api/crawl`

**Content-Type:** `application/json`

### Body Parameters

<ParamField body="url" type="string" required>
  The origin URL to begin spidering.
</ParamField>

<ParamField body="depth" type="number" default={3}>
  The nested hop limit. `2` means trace all links on the root, and all links on *those* pages.
</ParamField>

<ParamField body="maxPages" type="number" default={50}>
  Failsafe limit to prevent recursive crawling explosions.
</ParamField>

<ParamField body="treeOnly" type="boolean" default={false}>
  Synchronous mode. The engine discovers the topology and returns the map, but stops before converting the pages.
</ParamField>

<ParamField body="async" type="boolean" default={false}>
  Enterprise mode. Forces the service to instantly return a `202 Accepted` and offload the crawl to background GPU workers.
</ParamField>

<ParamField body="webhook_url" type="string">
  Production integration. The 2md clusters will HTTP POST the final data to this webhook upon completion.
</ParamField>

## Response (Async Mode)

When hitting the API asynchronously, the system immediately drops the connection and hands you a job receipt.

### 202 — Accepted

```json
{
  "success": true,
  "job_id": "j_98fbc12a",
  "status": "running",
  "status_url": "/api/jobs/j_98fbc12a",
  "result_url": "/api/jobs/j_98fbc12a/result"
}
```

## Production Examples

For building a continuous pipeline that ingests third-party documentation directly into Pinecone:

<CodeGroup>

```python Python (Webhook Driven)
import requests

# Kick off the massive crawl async
requests.post(
    "https://api.2md.traylinx.com/api/crawl",
    json={
        "url": "https://docs.anthropic.com",
        "depth": 5,
        "maxPages": 500,
        "async": True,
        "webhook_url": "https://api.yourstartup.com/webhooks/2md" # We call you back here
    }
)
print("Pipeline initiated. Awaiting webhook callback...")
```

```javascript Node.js (Polling)
const response = await fetch('https://api.2md.traylinx.com/api/crawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    url: 'https://docs.anthropic.com',
    async: true
  })
});

const { job_id } = await response.json();

// Poll the status URL until completed
const checkStatus = setInterval(async () => {
    const status = await fetch(`https://api.2md.traylinx.com/api/jobs/${job_id}`);
    const data = await status.json();
    if(data.status === 'completed') {
        console.log("Archive ready at:", data.download_url);
        clearInterval(checkStatus);
    }
}, 5000);
```
