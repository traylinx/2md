## Overview

The `POST /api/file2md` endpoint breaks binary format restrictions, allowing your extraction pipelines to process uploaded blobs (e.g. PDFs and Images) or raw cloud URLs natively into Markdown strings. 

**Base URL:** `https://api.2md.traylinx.com`

## Formatting Engines
The File2MD router inspects the incoming MIME type and delegates it automatically:
- **Optical Character Recognition (OCR)** — PDF, DOCX, XLSX
- **SwitchAI Vision Engine** — PNG, JPG, WEBP
- **SwitchAI Whisper Audio** — MP3, MP4, YouTube Links

## Request

`POST /api/file2md`

**Content-Type:** `multipart/form-data`

### Form Fields 

<ParamField body="file" type="blob">
  The direct binary file blob (if executing an upload).
</ParamField>

<ParamField body="url" type="string">
  The direct remote link to the binary asset (if avoiding local uploads).
</ParamField>

<ParamField body="apiKey" type="string">
  Your SwitchAI API Key. (Required for Vision/Whisper execution on images/video).
</ParamField>

<ParamField body="enhance" type="boolean" default={true}>
  When `true`, visually torn OCR dumps (e.g. from complex PDF columns) are forced through a secondary AI pass to reconstruct coherent, valid Markdown semantics seamlessly.
</ParamField>

## Response (JSON Format)

Include `?format=json` exactly on the endpoint boundary to retrieve cleanly structured serialized outputs.

```json
{
  "success": true,
  "files": {
    "pages/page_1.md": "# Title Header\n\nPage one text...",
    "full_document.md": "# Title Header\n\nPage one text...\n\nPage two text...",
    "metadata.json": "{\"original_filename\":\"report.pdf\",\"total_pages\":2}"
  }
}
```

## Production Examples

<CodeGroup>

```python Upload Binary PDF
import requests

with open('Q4_Earnings.pdf', 'rb') as f:
    res = requests.post(
        "https://api.2md.traylinx.com/api/file2md?format=json",
        files={'file': f},
        data={'enhance': 'true'}
    )
print(res.json()['files']['full_document.md'])
```

```python Translate YouTube to MD
import requests

res = requests.post(
    "https://api.2md.traylinx.com/api/file2md?format=json",
    data={
        'url': 'https://youtube.com/watch?v=dQw4w9WgXcQ',
        'apiKey': 'sk-your-switchai-token'
    }
)
# Returns the fully transcribed audio timeline natively formatted in Markdown
print(res.json()['files']['full_document.md'])
```

```bash Provide Cloud Remote Link
curl -X POST "https://api.2md.traylinx.com/api/file2md?format=json" \
  -F "url=https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf" \
  -F "enhance=true"
```
