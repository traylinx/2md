## Overview

The `POST /api/agentify` endpoint merges Markdown extraction with LLM cognition. It crawls a target path and automatically writes a routing index (`SKILL.md`) that explicitly teaches remote agents like Cursor and Windsurf how to synthesize the codebase correctly.

**Base URL:** `https://api.2md.traylinx.com`

<Note>
**Authentication:** Generating custom topologies requires LLM inference on the backend. You must pass a valid SwitchAI token (`apiKey`) in the payload to invoke this orchestration securely.
</Note>

## Request

`POST /api/agentify`

**Content-Type:** `application/json`

### Body Parameters

<ParamField body="url" type="string" required>
  The root target to compile.
</ParamField>

<ParamField body="apiKey" type="string" required>
  Your SwitchAI API Key (or OpenAI key for fallback modeling). Keys are scrubbed immediately post-response.
</ParamField>

<ParamField body="targetAgent" type="string" default="web">
  Optimizes the markdown dialect and routing language output:
  - `web` — For custom GPTs and remote Assistants.
  - `local` — For IDEs like Cursor analyzing the bundle locally.
</ParamField>

<ParamField body="async" type="boolean" default={false}>
  Returns a `job_id` instantly, permitting background synthesis. Required for targets generating over 20 endpoints.
</ParamField>

## Response 

```json
{
  "success": true,
  "files": {
    "SKILL.md": "# Agent Routing Table\n\n## Capability: Authentication...",
    "llms.txt": "# SDK Framework\n\n> AI-ready framework rules...",
    "references/auth.md": "# Authentication API\n\n...",
    "references/deploy.md": "# Deployment Pipelines\n\n..."
  }
}
```

The output yields a fully normalized `files` dict mapping filenames to their literal text string blobs. A system script can iterate over this object and dump it natively to disk in milliseconds.

## Production Examples

Compile Stripe's native documentation into a local Skill package to assist your developers natively inside Cursor without them ever leaving the IDE.

<CodeGroup>

```python Terminal Download Script
import requests
import os

res = requests.post(
    "https://api.2md.traylinx.com/api/agentify",
    json={
        "url": "https://docs.stripe.com",
        "apiKey": "sk-your-switchai-token",
        "targetAgent": "local"
    }
)
data = res.json()

# Write the bundle natively to your local repository
for filename, content in data['files'].items():
    os.makedirs(os.path.dirname(filename) or '.', exist_ok=True)
    with open(filename, 'w') as f:
        f.write(content)
print("Stripe SKILL.md agent compiled locally!")
```
