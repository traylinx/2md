The Convert page is the fastest way to turn a single URL into clean Markdown. It is designed to extract only the semantic content that matters to Large Language Models (LLMs), discarding navigation noise, tracking scripts, and heavy styling.

## Accessing the Convert console

Navigate directly to **[https://2md.traylinx.com](https://2md.traylinx.com)** in your browser. The Convert dashboard is the primary workspace.

<Note>
Using the UI is completely free and requires no API keys or configuration for standard web content.
</Note>

## Converting a page

<Steps>
  <Step title="Paste a URL">
    Enter any target URL into the input field. The system automatically bypasses standard captchas and handles geographic blocks.
  </Step>
  <Step title="Configure extraction">
    Use the toggles below the input box to configure real-time extraction rules:

    | Option | Description |
    |--------|-------------|
    | `Images` | Downloads embedded images and rewrites their Markdown paths so they are preserved in context. |
    | `Metadata` | Extracts title, author, and description metadata into YAML front matter for RAG routing. |
    | `Screenshots` | Captures a high-resolution visual snapshot of the DOM at the exact moment of extraction. |
  </Step>
  <Step title="Click Convert">
    The processing engine intelligently assigns the optimal extraction method (Native HTTP, Static Parsing, or full Headless Browser). Real-time progress logs stream to your client.
  </Step>
  <Step title="Copy and Use">
    The final Markdown is instantly rendered on screen. Copy the result directly into Claude, ChatGPT, or Cursor for immediate prompting context.
  </Step>
</Steps>

## Quality and Token Metrics

After conversion, a metrics card displays the exact token efficiency of the extraction natively comparing raw DOM nodes against the polished output:

| Metric | LLM Significance |
|------|-------------|
| HTML vs MD Tokens | Shows the percentage of context window saved (often exceeding 80%). |
| Headings & Links | Validates that structural semantics survived the extraction. |
| Cache Hit | Indicates if the edge network instantly served a pre-computed extraction. |

## Real-World SaaS Use Cases

- **Debugging Documentation**: Convert a multi-tabbed API reference page into raw Markdown to feed to an instance of `o1-preview` for debugging.
- **Bypassing Paywalls**: Pass a URL through the headless browser queue to extract an article cleanly behind cookie walls without pasting raw text blocks.
- **Retrieval Augmented Generation (RAG)**: Extract precise HTML shards into semantic Markdown to safely ingest into a Pinecone vector database.

## URL-Prepend Shortcut

For instantaneous conversion without leaving your workflow or opening the dashboard, append our proxy directly in front of your target URL:

```text
https://2md.traylinx.com/https://example.com/article
```

This bypasses the UI and natively writes the extracted Markdown directly back to your browser window.

<Tip>
Adding `?format=json` to the prepend URL (e.g., `https://2md.traylinx.com/https://example.com?format=json`) returns a fully structured payload containing metadata boundaries, token counts, and the native markdown string for programmatic parsing.
</Tip>
