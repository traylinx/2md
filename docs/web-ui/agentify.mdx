Agentify is a specialized aggregation pipeline that goes beyond raw Markdown extraction. It natively sequences a site through our secondary LLM processing router to output machine-readable intelligence standards (`SKILL.md` and `llms.txt`).

These packages allow advanced coding agents (Cursor, Windsurf, specialized GPTs) to autonomously navigate, read, and comprehend target SDKs and manual documentation without manual human prompting.

## Accessing Agentify

Navigate to **[https://2md.traylinx.com/agentify](https://2md.traylinx.com/agentify)** via the dashboard.

<Info>
The Agentify orchestration requires heavy LLM cognitive routing to create dynamic mapping tables. You must provide a valid SwitchAI API Key (or OpenAI key) locally in the browser to power the synthesis step. No keys are ever retained on the backend servers.
</Info>

## What is a Skill Bundle?

When the Agentify workflow completes against a target site, it issues a compressed bundle formatted strictly for AI comprehension:

| File | Agent Usage |
|------|----------|
| `SKILL.md` | The universal routing table mapping abstract concepts ("How to authenticate") to exact hardcoded file paths so the agent knows exactly what to read. |
| `llms.txt` | Discovers the root structure of the website using the [llms.txt protocol](https://llmstxt.org). |
| `references/*.md` | The actual pristine Markdown extractions used as ground-truth memory. |

## Real UI Workflow

<Steps>
  <Step title="Enter Key & Target">
    Enter your BYOK key into the secure local storage prompt. Input the base URL of the exact framework or SDK documentation you want your agent to learn.
  </Step>
  <Step title="Discover SDK Endpoints">
    Execute a discovery spider. Deselect irrelevant pages like "Community Guidelines" or "Blog".
  </Step>
  <Step title="Start Synthesis">
    Click **Agentify**. The 2md clusters convert the endpoints perfectly, while the SwitchAI LLM concurrently reads their content and drafts the `SKILL.md` routing table pointing abstract SDK tasks directly to the files.
  </Step>
  <Step title="Agent Console Review">
    Prior to downloading, the dashboard launches the **Agent Console**. This resembles a standard IDE where you have exact control to modify the `SKILL.md` instructions, delete false positives, or manually append additional context.
  </Step>
  <Step title="Download Bundle">
    Download the final zip bundle directly to your local machine.
  </Step>
</Steps>

## Drop-In Cursor Integration

When the zip is completed:
1. Extract the folder alongside your source code within your IDE.
2. Inside Cursor Composer, reference `@SKILL.md` directly.
3. Because all endpoints are logically chained, Cursor inherently knows how to deploy the external API architecture without encountering hallucinated logic!
