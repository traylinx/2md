## What is 2md?

**2md** is a high-fidelity HTML-to-Markdown converter built for developers, AI engineers, and content teams who need reliable, structured Markdown from any web source.

It combines a headless Chrome rendering engine with intelligent content extraction to produce clean, structured Markdown — even from complex SPAs, JavaScript-heavy sites, and behind Cloudflare WAF protection.

## How it works

Every conversion runs through a 5-step pipeline:

```
URL → Fetch → Extract → Convert → Download → Polish → page.md
```

| Step | Tool | What it does |
|------|------|-------------|
| **Fetch** | Puppeteer + Stealth | Renders the page in headless Chrome, bypasses WAF/Cloudflare |
| **Extract** | Readability.js → Trafilatura | Strips boilerplate; falls back to Python if quality is low |
| **Convert** | Turndown.js + GFM | Converts clean HTML to GitHub-Flavored Markdown |
| **Download** | Node fetch | Downloads referenced images, rewrites paths |
| **Polish** | Custom rules | Normalizes whitespace, adds YAML front matter, scores quality |

## Access modes

2md is available through three interfaces:

- **CLI** — `./bin/html2md` and interactive mode (`npm run interactive`)
- **REST API** — `POST /api/convert`, `/api/batch`, `/api/crawl`, `/api/agentify`, `/api/file2md`
- **Web UI** — full SPA at `http://localhost:8222` (or `https://2md.traylinx.com`)

<Note>
  Advanced features — **Agentify** and **File2MD** (PDF/media with Vision and Whisper) — are only available via the Web UI and REST API. The CLI covers single URL, batch, and crawl conversions.
</Note>

## Tech stack

| Component | Tool |
|-----------|------|
| Page rendering | Puppeteer (headless Chrome) |
| Content extraction | Readability.js + Trafilatura (Python fallback) |
| HTML → Markdown | Turndown.js + GFM plugin |
| API server | Express.js |
| Concurrency | p-queue |
| Job queuing | BullMQ + Valkey (Redis) |
| Storage | AWS S3 |
| Container | Docker (Node 22 + Chrome + Python 3) |
